#FIGURE2 Phylogenetic Tree

The phylogenetic tree for the HOT5 strains B08, C03, B10, E06, and F03 was generated in KBASE.

The following genomes were included in the "Insert Genome Into SpeciesTree - v2.2.0"
E06-rastk
B10-rastk
F03-rasttk
C03-rastk
B08-rastk
Rhodobacter sp LPB 0142
Pelagicola litoralis
Phaeobacter Inhibens DSM 16374
Dinoroseobacter shibae DFL 12 DSM 16493
Alteromonas macleodii (v1)


*The Hot5 genomes were annotated from their assemblies using the rasttkv1 app in KBASE ""
*The additional reference genomes from the strain library in Valiya Kaladi et al 2026 were imported into the KBASE narrative from their NCBI ACCESSION numbers.

File outputs for the SpeciesTree can be found in this folder ("new-tree.newick" and "new-tree-labels.newick") as well as the script in R for plotting from the files and the high res images.

The Narrative for generation of this tree can be found here: https://narrative.kbase.us/narrative/206763
